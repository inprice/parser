Customise your MacBook Air — Space Grey.

Hardware

Memory

Memory

MacBook Air comes standard with 8GB of high-performance 2133MHz memory and can be expanded to 16GB. The more memory your Mac has, the more apps you can run simultaneously and the better they will perform. If you’re planning to run multiple professional applications simultaneously, it’s a good idea to maximise your memory.

Please note that the memory is built into the computer. If you think you may need more memory in the future, consider upgrading when you purchase your MacBook Air.

Storage

Storage

This MacBook Air model comes with 256GB of SSD storage. For maximum capacity, you can configure it with up to 1.5TB of storage, giving you more space to store your documents, photos, music and apps, as well as extensive video libraries and large data files.

Please note that the storage is built into the computer. If you think you may need more storage capacity in the future, consider upgrading when you purchase your MacBook Air.

Keyboard Language

Keyboard Language

You can configure your MacBook Air with one of the keyboard languages listed.

macOS lets you choose your preferred language throughout the OS, including for apps, the menu bar, dialogue boxes, and reading and writing text. Select any of the languages supported by macOS during the setup process or at any time using System Preferences.

Note: Your computer comes with documentation appropriate for the country in which you purchased it.
